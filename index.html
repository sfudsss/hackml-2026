What is HackML?

HackML 2026 is a collaborative, fast-paced machine learning competition. Teams will build and submit ML models to solve real-world datasets, competing for the highest accuracy on a hidden test set.

How It Works

  2. Join the private Kaggle competition
  3. Train models and submit predictions
  4. Climb the live leaderboard
